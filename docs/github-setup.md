# GitHub Setup

## Create The Repository

1. Go to GitHub and create a new repository named `smarttax-ai`.
2. Keep it public if you want recruiters to see it.
3. Do not initialize with a README because this project already has one.

## Push From Local

```bash
git init
git add .
git commit -m "Build SmartTax AI full-stack project"
git branch -M main
git remote add origin https://github.com/Gowtham0408/smarttax-ai.git
git push -u origin main
```

## Suggested Repository Description

Cloud-ready full-stack tax assistant using Spring Boot, React, Python OCR, MySQL, AWS S3/Lambda, and CI/CD.
